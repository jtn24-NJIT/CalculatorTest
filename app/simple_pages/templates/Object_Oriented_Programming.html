Polymorphism: The condition of having multiple forms, in programming polymorphism is the same function name being utilized for different types. Being in-built or user defined, polymorphism allows for easier creation of multiple classes as well as usage of functions upon different data types. For example, the len function in Python can be used both to count the number of characters in a string, as well how many items there is in a list. When used for classes/objects, polymorphism allows for inheritance; which is used to share and allow for overwriting of common features.
